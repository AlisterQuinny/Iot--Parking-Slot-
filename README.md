# VisionSlot 🚗

VisionSlot is an IoT-based Smart Parking System made using Arduino UNO.

## Features
- Automatic barrier opening
- Parking slot detection
- LCD display for slot status
- IR sensor-based vehicle detection
- Servo motor gate control

## Components Used
- Arduino UNO
- IR Sensors
- Servo Motors
- 20x4 I2C LCD
- Jumper Wires

## Technologies
- Arduino IDE
- Embedded C++
